## Install

```sh
curl -fsSL https://cli.craftybase.dev/install | bash
```

With Homebrew:

```sh
brew install craftybase/tap/craftybase
```

With Go:

```sh
go install github.com/craftybase/craftybase-cli/cmd/craftybase@latest
```
